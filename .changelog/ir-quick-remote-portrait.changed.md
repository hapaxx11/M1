**Infrared: quick-remote button grid now automatically uses portrait (remote-control) orientation** — TV, AC, Audio, Projector, Fan and LED quick-remote grids now switch to portrait mode on entry and restore the previous orientation on exit. The grid layout adapts dynamically to display dimensions: 2-column layout with correctly-sized cells in portrait (64×128), original 3-column layout in landscape. File-browser screens (Right: browse device) temporarily switch to landscape while the list is shown.
