**Sub-GHz: decode scene styling and actions parity with Momentum** —
  Decode results list views (Read Raw → More → Decode and Saved → file →
  Decode) now use the standard menu styling: `m1_menu_font()` /
  `m1_menu_item_h()` / `M1_MENU_TEXT_W` highlight, scrollbar, and
  user-configurable text size.  Detail view adds Send (OK) and Save (DOWN)
  actions so decoded signals from RAW captures can be transmitted or saved
  as .sub/.sgh key files.  TE display prefers the protocol-specified value
  over the observed average for consistency with Momentum.
